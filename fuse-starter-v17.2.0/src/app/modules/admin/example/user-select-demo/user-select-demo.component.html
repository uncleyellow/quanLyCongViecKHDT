Usage Instructions

Single Selection: Click on a user to select them. Only one user can be selected at a time.

Multiple Selection: Click on multiple users to select them. Selected users appear as chips above the input.

Search: Type in the input field to filter users by name, email, or user type.

Remove Selection: Click the 'x' button on user chips to remove them from selection.

Clear All: Click the 'clear_all' button to remove all selected users.

API Information

Endpoint: GET /api/v1/users/basic-list

Authentication: Requires Bearer token

Role-based Access: Users see different lists based on their role and department

Response Format: Returns basic user information (id, name, email, avatar, userType)
